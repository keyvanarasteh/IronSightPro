# Support

Need help with IronSight? We are here to help!

## Seeking Support

When looking for support, please consider the following channels:

1. **GitHub Discussions:** The best place for support questions is the [GitHub Discussions](https://github.com/keyvanarasteh/IronSight/discussions) tab. Look for the "Q&A" category.
2. **Issue Tracker:** If you're confident you have found a bug or want to request a new feature, please use the [Issue Tracker](https://github.com/keyvanarasteh/IronSight/issues). Avoid using the issue tracker for support questions.
3. **Documentation:** Check our [Wiki](../../wiki) or the existing `README.md` files for answers.

## Providing Support

We love when our community helps out! If you want to help, you can browse [unanswered questions in Discussions](https://github.com/keyvanarasteh/IronSight/discussions?discussions_q=is%3Aunanswered+category%3AQ%26A) or tackle [open issues labelled "help wanted"](https://github.com/keyvanarasteh/IronSight/issues?q=is%3Aissue+is%3Aopen+label%3A"help+wanted").

*Note: IronSight is maintained as part of an academic and open-source effort at İstinye University. Responses are provided on a best-effort basis.*
