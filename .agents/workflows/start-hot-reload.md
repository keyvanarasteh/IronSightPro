# Dioxus Hot-Reloading

This workflow starts the Dioxus Desktop UI with experimental Rust hot-reloading enabled.

// turbo
1. Start the development server using the Dioxus CLI (`dx`):
```bash
dx serve --bin ironsight-ui --platform desktop --package ironsight-ui
```
